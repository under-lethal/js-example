JS Identifiers

All Js variables must be identified with unique names.
These unique names are called identifiers
Identifiers can be short names(like x and y) or more descripive names (age, sum, totalVolume).

The general rules for constructing names for variables (unique identifiers) are:

  • Names can contain letters, digits, underscores, and dollar signs
  • Names must begin with a letter
  • Names can also begin with $ and _
  • Names are case sensitive (y and Y are different variables)
  • Reserved words (like JavaScript keywords) cannot be used as names

The Assigment Operator

In JS, the equal sign (=) is an "assigment" operator, not an "equal to" operator.

In JavaScript, however, it makes perfect sense: is assigns the value of x + 5 to x.

The "equal to" operator is written like == in JavaScript.

Declaring (Creating) JavaScript Variables

Creating a variable in JavaScrip is called "declaring" a variable.
You declare JS variable with the var keyword:

var carName;

After the declaration, the variable has no value. (Tehnicaly is has the valus of ,undefined)
To assign a value to the variable use the equal sign:

carName = "Volvo"

And assign a value to the variable when you declare it:
var carName = "Volvo"
