JavaScript Identifiers

Identifiers are names.

In JavaScript, identifires are used to name variables (and keywords, functions, and labels).
The rules for legal names are much the same in most rograming languages.
In JavaScript, the first character must be a letter, an underscore (_), or a dolar sign ($).
Subsequent characters may be letters, digits, underscores, or dollar signs.

Numers are not allowed as the first character.
This way JavaScript can easily distinguish identifiers drom numbers.

JavaScript is Case Sensitive

All JavaScript identifiers are case sensitive.
