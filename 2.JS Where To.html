The <script> Tag

In HTML, JavaScript code must inseretd betwen < script > and tags.

JavaScript Functions an Events

A JavaScript function is a block of JavaScript code, that can be executed when "called" for. For example, a function can be called when an event occurs, like when the user clicks a button.

JavaScript in < head > or < body >

You can place any number of scripts in an HTML document.
Scripts can be placed in the < body >, or in the < head > section of an HTML page, or in both. Placing scripts at the bottom of the < body > element improves the display speed, because script compilation slows down the display.

External JavaScript

Scrpts can be also be placed in external files:
External scripts are practical when the same code is used in many different web pages.
JavaScript files have the file extenstion .js To use an external script, put the name of the script file in the src (source) attributte of a < script > tag.

You can place an external script reference in < head > or < body > as you like.
The script will behave as if it was located exactly where teh < script > tag is located. External scripts cannot contain < script > tags.

External JS Advantages

PLacing scripts in external files has some advantages:

  • It separates HTML and code
  • It makes Htmls and JavaScript easier to read and mantain
  • Cached JavaScript files cand speed up page loads
